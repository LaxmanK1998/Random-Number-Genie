# Number Genie 🎲

A clean, responsive native Android application that generates random numbers, wrapped in a custom mathematical textbook interface. This project was built using modern Android development practices, featuring **Kotlin** and **Jetpack Compose** for a fully declarative user interface.

## ✨ Features
* **Dynamic Generation:** Instantly generates a random integer between 1 and 100 with a single tap.
* **Custom Textbook UI Theme:** Features a unique, low-contrast overlay of classical mathematical formulas and equations (`e^(iπ) + 1 = 0`, matrix transformations, etc.) styled to look like an academic textbook background.
* **Modern Material 3 Design:** Uses high-contrast interactive elements, including a striking custom crimson action button.
* **Adaptive Graphics:** Includes custom asset integration for modern Android launcher icon safe zones.
